# Movie API

# Tech Stack

- NestJS (framework)
- PassportJS (for authentication)
- bcrypt (for password hashing)
- TypeORM
- MySQL (DB)
- NodeMailer (for sending mail)
- Jest (UT)

## Run Docker

- docker compose up -d
- http://localhost:3000

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test
```
